# Pipeline Thinking Foundations

## Overview

This repository contains my work for the "Pipeline Thinking Foundations" exercise. 

The goal of this exercise is to develop a data engineering mindset by analyzing different business scenarios and identifying:

- The data source
- The data problems
- The required transformations
- The expected clean output

The focus is on pipeline thinking rather than coding.

## Repository Structure

- README.md — Project overview
- answers.md — Analysis of the four scenarios
- reflections.md — Personal reflection on what I learned

## Objective

The purpose of this exercise is to practice thinking like a data engineer before building actual pipelines in Python, Spark, or Databricks.
